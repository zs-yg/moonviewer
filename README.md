# 月光查看器 (Moon Viewer)

一个使用PySide6开发的现代化图片查看器，支持PNG格式和多文件夹管理。

## 功能特性

- ✅ 支持PNG、JPG、JPEG、BMP、GIF、TIFF、WEBP格式
- ✅ 多文件夹管理：可以添加和管理多个图片文件夹
- ✅ 现代化GUI界面：暗色主题，响应式布局
- ✅ 图片浏览：上一张/下一张导航
- ✅ 缩放功能：放大、缩小、重置缩放
- ✅ 图片信息显示：文件名、路径、大小、尺寸、格式等
- ✅ 缩略图预览
- ✅ 文件树状视图：浏览文件夹结构
- ✅ 文件列表：显示当前文件夹中的所有图片
- ✅ 状态栏：显示操作状态和图片信息

## 系统要求

- Python 3.12+
- PySide6
- Pillow (用于图片处理)

## 安装

1. 克隆或下载本项目
2. 安装依赖：

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行主程序：

```bash
python main.py
```

2. 使用界面：
   - **打开图片**：通过菜单栏或工具栏打开单个图片文件
   - **打开文件夹**：通过菜单栏或工具栏打开包含图片的文件夹
   - **添加文件夹**：通过左侧面板的"添加文件夹"按钮添加更多文件夹
   - **浏览图片**：在左侧文件列表中选择图片，或使用导航按钮浏览
   - **缩放图片**：使用工具栏的放大、缩小、重置缩放按钮
   - **查看信息**：右侧面板显示当前图片的详细信息

## 界面说明

- **左侧面板**：
  - 文件夹树：浏览系统文件夹结构
  - 添加文件夹按钮：添加新的图片文件夹
  - 文件列表：显示当前选中文件夹中的所有图片文件

- **中间面板**：
  - 图片显示区域：显示当前选中的图片
  - 导航按钮：上一张/下一张

- **右侧面板**：
  - 图片信息：显示图片的详细信息
  - 缩略图：显示当前图片的缩略图

- **菜单栏**：
  - 文件：打开图片、打开文件夹、退出
  - 查看：放大、缩小、重置缩放

- **工具栏**：
  - 常用操作的快捷按钮

- **状态栏**：
  - 显示当前操作状态和图片信息

## 快捷键

- `Ctrl+O`：打开图片
- `Ctrl+Shift+O`：打开文件夹
- `Ctrl++`：放大
- `Ctrl+-`：缩小
- `Ctrl+0`：重置缩放
